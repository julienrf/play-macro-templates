@title

This Web site demonstrates the utilization of Scala macros to dynamically call Play! templates. The pages you see are rendered using a template that is dynamically chosen according to the request URL path. In the same way, the navigation menu is dynamically generated from the list of pages.

The code of this experiment is available on GitHub. If you want to try it, just clone the repository:

$ git clone git@@github.com:julienrf/play-macro-templates.git

And run the application:

$ cd play-macro-templates; sbt run

If you are curious, feel free to explore the pages explaining the page rendering mechanism and the menu construction mechanism.
